================================================================================
HOW TO USE
================================================================================

Local Development (Tiers 0-2):

1. Setup:
   pip install -r requirements.txt
   cp .env.example .env
   # Add GEMINI_API_KEY to .env

2. Test:
   python test_tier0.py
   python test_tier1.py
   python -m wealth_agent.eval

3. Run:
   adk web
   # Visit: http://localhost:8000
   # Ask: "What's my net worth?" or "Rebalance to 60/40"

Cloud Deployment (Tier 3):

1. Setup:
   cd terraform
   cp terraform.tfvars.example terraform.tfvars
   # Edit: project_id, gemini_api_key

2. Deploy:
   terraform init
   terraform apply
   # Output: Public Cloud Run URL

3. Use:
   curl https://wealth-agent-xyz.a.run.app/dev-ui/
   # Auto-scales, 0-10 instances, fully managed
